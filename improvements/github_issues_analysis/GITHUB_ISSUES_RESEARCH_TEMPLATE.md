**Category Definitions:**
- **Missing:** Feature/concept exists but has no documentation
- **Outdated:** Documentation exists but references old/deprecated info
- **Unclear:** Documentation exists but users find it confusing
- **Findability:** Information exists but users can't find it
- **Context:** Documentation assumes knowledge users don't have
